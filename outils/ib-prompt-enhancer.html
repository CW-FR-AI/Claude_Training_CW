IB Prompt Enhancer.

Améliore silencieusement vos prompts IB casual en versions deal-ready, puis exécute. Vous obtenez la qualité d'un prompt rigoureux sans avoir à le formuler — Claude réécrit en interne et délivre directement le résultat.

— Ce qu'il fait

Un Skill qui travaille en coulisses

Vous écrivez : « fais-moi un panorama du SaaS RH français ». Claude détecte une tâche IB et déclenche le Skill. En interne, il réécrit votre prompt en version rigoureuse :

  • Périmètre géographique et sectoriel précisé
  • Sources prioritaires (Pappers, S&P Global, presse spécialisée)
  • Format de sortie attendu (segmentation, dynamiques, acteurs, transactions récentes)
  • Contraintes de qualité (sourcing systématique, pas de chiffres non vérifiés)
  • Convention de format Clearwater

Puis il exécute. Vous voyez juste le résultat — un livrable de qualité d'un prompt enhanced, sans avoir à le rédiger vous-même.

Pourquoi c'est utile

Pour un VP qui doit produire vite, c'est plusieurs minutes économisées par requête. Pour un analyste qui ne maîtrise pas encore les standards de prompting IB, c'est l'assurance d'un output deal-ready sans courbe d'apprentissage.

— Déclenchement

Comment l'invoquer

Skill silencieuse — vous n'avez rien à faire. Elle se déclenche automatiquement sur toute requête IB / M&A / deal / diligence.

Types de requêtes qui activent le Skill

  • Panorama / overview sectoriel
  • Recherche de cibles ou d'acquéreurs (long-list)
  • Profil de société, fact sheet
  • Analyse de CIM, deck management, comptes
  • Comparables (peers, transactions)
  • Préparation de pitch, mémo, IC
  • Toute requête mentionnant « M&A », « deal », « valuation », « due diligence », « screening »

Vous n'avez pas besoin de soigner votre formulation — c'est même le point. Tapez naturellement, le Skill fait le travail derrière.

— Cas d'usage

Avant / après concret

Votre prompt casual
« Trouve-moi des cibles dans le SaaS RH français entre 10 et 30m de CA »
Prompt enhanced (en interne)
Identifie 15 cibles potentielles dans le SaaS RH FR, CA 10-30m€. Critères : indépendance capitalistique, croissance organique > 15%, équipe dirigeante stable. Sources : Pappers (financiers FR), Grata (signaux M&A), presse spécialisée. Format tabulaire : nom, géo, CA, EBITDA, actionnariat, signal M&A, source. Vérifier l'existence de chaque cible avant inclusion.

Vous récupérez le résultat — un tableau structuré de 15 cibles avec sources. Sans avoir eu à formuler les contraintes de qualité.

— Pièges & limites

À garder en tête

Quand préférer l'Architect Si vous voulez voir et stocker le prompt enhanced pour le réutiliser, ou pour un cas particulièrement complexe, utilisez plutôt IB Prompt Architect avec @architect — qui produit le prompt sans exécuter, donc vous pouvez le copier dans un Project.
  • Sortie déterministe. Le Skill choisit le format de sortie en fonction de la requête. Si vous voulez un format spécifique, précisez-le dans votre prompt initial — le Skill respectera la contrainte.
  • Volume. Pour des requêtes très lourdes (long-list de 100 cibles, panorama de 6 segments), le Skill ne change pas la nature de la limite — découpez en plusieurs messages.
  • Confidentialité. Le Skill applique les standards IB de confidentialité (pas de mention publique de données sensibles). Mais pour les mandats sous NDA stricte, travaillez depuis un Project dédié — voir Confidentialité.
