— Ce qu'il fait

Sources curées pour le M&A

Dès que vous lancez une recherche dans un contexte M&A / IB / deal / corporate finance, le Skill dirige Claude vers une liste de sources de référence préselectionnées par Clearwater. Pas de wikipedia, pas de sources non vérifiées, pas de comparateurs grand public.

Les 7 catégories de sources curées

  • Advisory firms — publications et rapports des grandes boutiques M&A et IB de référence
  • Strategy consultants — McKinsey, BCG, Bain, Roland Berger, et autres cabinets de stratégie
  • Macro / regulatory bodies — INSEE, Banque de France, Eurostat, BCE, AMF, autorités sectorielles
  • Business press — Les Échos, Wall Street Journal, Financial Times, Bloomberg, Reuters
  • Trade press — presse spécialisée par secteur (Capital Finance, Décideurs, M&A Magazine)
  • Federations — fédérations professionnelles par secteur (FEVAD, Syntec, etc.)
  • PE / debt / investor ecosystem — France Invest, Invest Europe, PitchBook, Mergermarket, presses PE

Citations conformes au format IB

Le Skill produit systématiquement des citations en footnote + une bibliography en fin de document. Format banker, pas format académique — directement utilisable dans un mémo, un pitch, ou un livrable client.

— Déclenchement

Comment l'invoquer

Skill silencieuse. Se déclenche automatiquement dès qu'une recherche M&A / IB est détectée.

Types de requêtes qui activent le Skill

  • Recherche sur une société, une cible, un fonds, une boutique advisory
  • Panorama / overview sectoriel
  • Long-list de cibles ou d'acquéreurs
  • Actualité ou news sur un acteur
  • « Cherche-moi », « parle-moi de », « tell me about », « research », « recent deals »

Le Skill se déclenche même sur des requêtes courtes ou implicites. Si vous tapez « LBO secteur santé France 2025 », le Skill comprend que c'est une recherche IB et oriente Claude vers France Invest, Capital Finance, Mergermarket plutôt que vers du contenu généraliste.

— Cas d'usage

Quand l'utiliser

Cas 1

Panorama sectoriel pour pitch

Tu prépares un pitch sur un nouveau secteur. Le Skill te ramène les sources fiables (consultants, fédérations) plutôt que des blogs marketing.

Cas 2

Recherche de deals comparables

Tu cherches des transactions M&A récentes dans un segment. Le Skill priorise Mergermarket, Capital Finance, presse PE.

Cas 3

Profil d'un acteur PE

Identification d'un fonds PE actif sur un segment. Le Skill mobilise France Invest, Invest Europe, PitchBook.

Cas 4

Veille réglementaire

Une question sur une régulation sectorielle (AMF, autorité de la concurrence). Le Skill ramène les sources officielles, pas des résumés journalistiques.

— Pièges & limites

À garder en tête

Vérifier les chiffres avant un livrable Le Skill cite des sources fiables mais ne garantit pas la fraîcheur ou l'exactitude de chaque chiffre. Pour tout chiffre clé qui finit dans un livrable client, recoupez avec la source primaire — surtout sur des données qui peuvent dater (multiples sectoriels, tailles de marché).
  • Liste curée non exhaustive. Si une source spécifique vous semble manquer (presse sectorielle pointue, base interne), citez-la explicitement dans votre prompt — le Skill l'intégrera.
  • Sources payantes. Certaines sources curées (Mergermarket, PitchBook, S&P Global) ne sont accessibles que via les MCP connectés. Vérifiez que le bon connecteur est activé pour les requêtes qui en dépendent.
  • Pour la recherche corporate FR pure (actionnariat, comptes), Pappers reste la source primaire — le Skill l'utilise automatiquement quand pertinent.
