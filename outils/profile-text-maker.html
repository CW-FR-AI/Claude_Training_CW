Profile Text Maker.

Rédige des profils de société et des one-pagers au format banque d'affaires, à partir d'un site web, de documents uploadés et de sources publiques. Factuel, structuré, source-disciplined.

Type
Skill · Invocation manuelle

Français + English
Modèle conseillé
Sonnet 4.6 ou Opus 4.7
— Ce qu'il fait

Profils sociétés en style banker

Profile Text Maker produit du texte structuré de profil société selon les conventions banque d'affaires. Pas un livrable PowerPoint formaté (pour ça, voir Clearwater Profile), mais le texte brut prêt à intégrer dans un livrable ou un mémo.

Ce que le Skill couvre

  • Présentation générale — activité, géographie, taille, positionnement marché
  • Ownership summary — actionnariat, holdings, contrôle
  • M&A track record — acquisitions et cessions historiques significatives
  • Management overview — dirigeants clés, parcours, ancienneté
  • Financials — chiffres clés sur 3-5 ans si disponibles publiquement

Discipline de sources

Le Skill cite ses sources systématiquement (footnote ou inline). Il refuse de produire un chiffre sans source vérifiable. Sur les sociétés françaises, il bascule sur Pappers en priorité. Sur les cotées, sur S&P Global.

— Déclenchement

Comment l'invoquer

Skill à invocation manuelle. Formulations qui le déclenchent :

  • « Fais-moi un profil de [société] »
  • « Rédige une fact sheet sur [société] »
  • « One-pager société sur [nom] »
  • « Company profile for [name] »
  • « Corporate fact sheet [nom] »

Pattern recommandé

  1. Donnez le nom exact de la société et son pays d'incorporation (évite les confusions sur les noms communs)
  2. Précisez les sections à inclure (toutes, ou un sous-ensemble selon vos besoins)
  3. Joignez des documents si vous en avez (CIM management presentation, site web, rapport annuel) — le Skill les utilisera en priorité
  4. Spécifiez la langue de sortie (FR par défaut, EN sur demande)
— Cas d'usage

Quand l'utiliser

Cas 1

Préparation de pitch

Profil d'un prospect ou client cible, à intégrer dans le pitch document. Format texte → import dans PowerPoint.

Cas 2

Long-list d'acquéreurs

Pour chaque acquéreur identifié, un profil court avec track record M&A et fit potentiel. Itérer le Skill par batchs de 5-10 acquéreurs.

Cas 3

Profil cible pour buy-side

Recherche cibles. Profile Text Maker structure l'info collectée pour évaluer rapidement plusieurs candidats.

Cas 4

Mémo préparation appel client

Avant un call sur un nouveau mandat, un profil rapide du prospect pour cadrer la discussion.

— Pièges & limites

À garder en tête

Vérifier les chiffres avant tout livrable client Le Skill cite ses sources mais peut hériter d'imprécisions des sources publiques (chiffres datés, fusion de filiales, etc.). Pour tout chiffre qui finit dans un document client, recoupez avec une source primaire.
  • Sociétés non cotées petites : Profile Text Maker manque de données fiables sur les très petites sociétés non cotées. Donnez-lui des documents internes (CIM, site web) pour qu'il s'appuie dessus.
  • Pas un livrable PowerPoint. Le Skill produit du texte. Pour le format Clearwater visuel (PowerPoint avec layout 3 colonnes, KPI gradient, charts), utilisez Clearwater Profile qui tourne sur Cowork.
  • Pour des dossiers sensibles, travaillez depuis un Project mandat — pas en chat libre. Voir Confidentialité.
