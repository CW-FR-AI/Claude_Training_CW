Clearwater Profile.

Génère un profil de société au format Clearwater officiel : PowerPoint A4 landscape, fonts Skeena (titres) + Bierstadt (corps), layout 3 colonnes avec KPI gradient sombre, graphique combo revenus, donut chart. Via Cowork.

— Ce qu'il fait

Slide profil société au format firme

Le Skill produit un slide PowerPoint complet de profil société, conforme aux standards visuels Clearwater. Pas du texte (pour ça, voir Profile Text Maker) — un livrable .pptx prêt à intégrer dans un pitch, un teaser, ou un mémo.

Format Clearwater appliqué automatiquement

  • Format A4 landscape (10.83" × 7.5")
  • Fonts : Skeena (titres + tagline) + Bierstadt (tout le reste)
  • Layout 3 colonnes — partie centrale large + colonne KPI à droite avec gradient sombre
  • Graphique combo revenus (bar + line) sur 3-5 ans
  • Donut chart ou pie chart pour la répartition (segments, géographies, etc.)
  • Palette Clearwater respectée (couleurs, accents)

Contenu produit

  • Présentation générale de la société (activité, géo, positionnement)
  • Chiffres clés (CA, EBITDA, croissance, employés)
  • Évolution financière (graphique 3-5 ans)
  • Répartition stratégique (segments, géo, clients selon disponibilité)
  • Actionnariat principal
  • Track record M&A si pertinent
— Déclenchement

Comment l'invoquer

Le Skill se déclenche dans Cowork (pas dans le chat web) quand vous demandez la production d'un profil société au format Clearwater.

Formulations qui déclenchent

  • « Fais-moi un profil société Clearwater sur [nom] »
  • « Company profile au format Clearwater »
  • « Profil de société format Clearwater »
  • « Fact sheet [nom] au format firme »
  • « One-pager Clearwater pour [société] »

Inputs à fournir

  1. Nom et site web de la société
  2. Documents internes si disponibles (CIM management presentation, deck investor) — Cowork les utilisera en priorité
  3. Langue de sortie (FR par défaut, EN sur demande)
  4. Niveau de détail souhaité (one-pager succinct vs slide riche)
— Cas d'usage

Quand l'utiliser

Cas 1

Profil cible pour CIM

Slide profil de la société cédée à intégrer dans la teaser ou l'executive summary du CIM.

Cas 2

Profil acquéreur pour mémo IC

Pour chaque acquéreur sérieux, un slide profil. Le Skill peut être lancé en batch sur 10-20 acquéreurs en Cowork.

Cas 3

Profil prospect pour pitch

Avant un pitch, produire le profil du prospect pour démontrer la connaissance du dossier.

Cas 4

Mise à jour de profils existants

Régénération annuelle d'une bibliothèque de profils sectoriels avec données mises à jour. Cowork fait le travail mécanique.

— Pièges & limites

À garder en tête

Cowork desktop nécessaire Le Skill tourne uniquement dans Cowork desktop (Mac / Windows). Pas accessible depuis le chat web ou mobile. L'app desktop doit rester ouverte pendant la génération.
  • Template Clearwater requis. Le Skill utilise le template officiel Clearwater (Template_slide.pptx) pour hériter du branding. Si le template est absent, le rendu sera dégradé.
  • Vérifier le rendu visuel. Cowork produit le .pptx mais ne valide pas la cohérence visuelle finale. Ouvrir le fichier dans PowerPoint pour QA avant utilisation client.
  • Sources des chiffres. Le Skill remplit les chiffres avec ce qu'il trouve. Pour un livrable client, vérifier les chiffres à la source avant envoi.
  • Pour le texte uniquement (sans format PowerPoint), préférer Profile Text Maker qui tourne en chat (plus rapide, plus itératif).
